# sh_session_exists
Given an array of tokens, returns those that exist.


```py
sh_session_exists(tkns: List[int]) -> List[int]
```

|Param|Description|
|--|---|
|tkns|Tokens to check|


## Returns
The tokens that exist. Each token in the `tkns` parameter that exist will be returned
