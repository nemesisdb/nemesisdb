# exist

```py 
async def exist(name: str) -> bool
```

|Param|Description|
|---|---|
|name|Name of the array|

Indicates if the given array exists.


## Array Type Differences
None


## Raises
- `ValueError` caught before query is sent
    - `name` is empty
