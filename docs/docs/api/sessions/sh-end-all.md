# SH_CLEAR
Deletes all sessions and their data.


## Response

`SH_CLEAR_RSP`

See the [response status](./../Statuses) page for status values.


|Param|Type|Meaning|
|:---|:---|:---|
|st|unsigned int|Status|
|cnt|unsigned int|Number of sessions deleted|


Possible status values:

- Ok
