# SH_INFO_ALL
Get information on all sessions.


No parameters.


## Response

`SH_INFO_ALL_RSP`

See the [response status](./../Statuses) page for status values.


|Param|Type|Meaning|
|:---|:---|:---|
|totalSessions|unsigned int|Number of session|
|totalKeys|unsigned int|Number of keys over all the sessions|
